# Payroll Management Microservices System

## Services
- Employee Service
- Attendance Service
- Payslip Service
- React Frontend

## Tech Stack
- Spring Boot
- MySQL
- React JS
- REST API
- Microservices

## Ports
- Employee Service → 8081
- Attendance Service → 8082
- Payslip Service → 8083
- React Frontend → 3000

## Features
- Add Employee
- Add Attendance
- Generate Payslip
- View All Employees
